# Assinaturas digitais para documentos AdL

Arquivos salvos a partir de imagens enviadas por Fábio no WhatsApp em 2026-06-16, para uso quando Fábio/Fabinho pedirem inclusão de assinatura em documentos oficiais.

## Arquivos

- `assinatura-fabio-de-luca.png`: assinatura de Fábio de Luca.
- `assinatura-fabio-oliviere.png`: assinatura de Fábio Oliviere / Fabinho.

## Uso

- Usar somente quando Fábio, Fabinho ou alguém autorizado pedir para inserir assinatura em documento específico.
- Não anexar automaticamente a documentos sem pedido explícito.
- Inserir no campo de assinatura mantendo proporção da imagem.
- Em PDFs HTML, referenciar por caminho local absoluto ou copiar para o diretório do template antes de renderizar.
